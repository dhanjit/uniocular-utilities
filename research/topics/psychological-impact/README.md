# Topic: Psychological and social impact

Vision loss is not only sensory; adjustment is psychological.

## Summary

Vision loss roughly doubles depression risk, the relationship is
bidirectional, and acceptance plus social support drive better adjustment.
Eye loss carries documented social-avoidance effects.

## Key findings

- ~2× depression risk; ~1 in 4 report anxiety/depression (CDC); suicide-risk
  link; bidirectional with stress; acceptance and support help — see
  [visual-impairment-mental-health](../../sources/visual-impairment-mental-health/).
- Acquired monocular vision carries serious psychosocial sequelae
  (depression, job loss) — see [kraut-2002-adaptation](../../sources/kraut-2002-adaptation/).
- A prosthesis improves general-health perception but not necessarily global
  stress — see [makrakis-2021-prosthesis-qol](../../sources/makrakis-2021-prosthesis-qol/).

## Implications for the project

- A frank, supportive emotional-adjustment section in the "first 90 days"
  manual; signpost mental-health resources. Not a substitute for clinical
  care.
- Respectful, person-first tone across all content.
