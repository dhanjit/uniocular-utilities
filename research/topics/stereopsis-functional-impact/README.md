# Topic: Functional impact of stereopsis loss

How much losing stereo vision actually costs, and at which tasks.

## Summary

Stereopsis matters most for fine motor tasks in near space under direct
viewing; it adds little when the task is mediated by a 2D screen. Distance
judgment beyond a few meters is essentially unaffected. Compensation is
possible but costs cognitive effort.

## Key findings

- Stereo-normal beats stereo-absent on direct-vision fine motor tasks;
  **depth** errors rise without stereo, **lateral** errors do not — see
  [stereopsis-surgical-performance-2017](../../sources/stereopsis-surgical-performance-2017/).
- Under **2D viewing**, the stereo advantage largely disappears — see
  [stereopsis-2d-3d-2014](../../sources/stereopsis-2d-3d-2014/). Big design
  implication: 2D screens level the field.
- Elite ball-sport performance does not require perfect stereopsis — see
  [barrett-2017-elite-athletes](../../sources/barrett-2017-elite-athletes/).
- Functional end-points (reading fatigue, slower fine motor) also appear in
  amblyopia — see [kelly-2025-amblyopia-hrqol](../../sources/kelly-2025-amblyopia-hrqol/).

## Task-difficulty (high level)

High impairment: near-field reaching/grasping, pouring, threading. Moderate:
gait/stairs, ball sports, driving in tight spaces. Essentially unaffected:
distance judgment > ~6 m, reading on a 2D screen (acuity). A fuller matrix
is in the [deep-research synthesis](../../deep-research-synthesis.md).

## Implications for the project

- Realistic, non-alarmist framing in manuals: name the genuinely hard tasks,
  reassure on the unaffected ones.
- Routing depth-critical work through 2D representations is a design lever
  (and a compatibility-standard principle).
