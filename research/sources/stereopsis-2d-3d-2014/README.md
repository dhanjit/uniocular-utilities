# Absence of stereopsis in 2D vs 3D surgical task (2014)

**Citation:** "The effects of absence of stereopsis on performance of a
simulated surgical task in two-dimensional and three-dimensional viewing
conditions." 2014.
**Identifiers:** PMID 25185439.
**Access:** Open access (<https://pmc.ncbi.nlm.nih.gov/articles/PMC4316921/>).
**Verification:** From search digest.

## Scope

Compared stereo-present vs. stereo-absent performance on a simulated
surgical task under 2D (video-assisted) vs. 3D viewing.

## Key points

- Under **2D viewing**, stereopsis confers **little advantage** — performance
  is relatively fair for stereo-absent individuals.
- The stereo advantage appears mainly under direct 3D viewing.

## Learnings for this project

- Strong design insight: routing depth-critical work through a **2D
  representation** (a screen) substantially levels the field for uniocular
  users. Supports the "reading/working on a 2D screen is largely unaffected"
  finding and the case for 2D fallbacks in the compatibility standard.
