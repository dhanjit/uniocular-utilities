# Depth from motion parallax and the motion/pursuit ratio (2014)

**Citation:** Nawrot M, et al. "Modeling depth from motion parallax with the
motion/pursuit ratio." *Frontiers in Psychology*. 2014. (See also "Visual
Depth from Motion Parallax and Eye Pursuit".)
**Access:** Open access (<https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4186274/>;
<https://pmc.ncbi.nlm.nih.gov/articles/PMC3348271/>).
**Verification:** From search digest.

## Scope

Models how the visual system extracts unambiguous depth (including
depth-sign, i.e. which object is nearer) from motion parallax.

## Key points

- The **smooth-pursuit eye-movement signal** is used to resolve depth-sign
  from parallax.
- Depth from parallax requires **translational** head/body movement, not
  rotation.

## Learnings for this project

- The mechanistic basis for the depth-cue trainer: parallax exercises should
  involve real translational head movement and visual pursuit.
