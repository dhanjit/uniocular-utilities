# Small-aperture monovision and the Pulfrich experience (2013)

**Citation:** "Small-Aperture Monovision and the Pulfrich Experience: Absence
of Neural Adaptation Effects." *PLOS ONE*. 2013.
**Identifiers:** PMID 24155881.
**Access:** Open access (<https://pmc.ncbi.nlm.nih.gov/articles/PMC3796532/>).
**Verification:** From search digest.

## Scope

Examined whether the Pulfrich effect induced by interocular differences
(monovision / small aperture) reduces with neural adaptation over time.

## Key points

- Interocular differences in brightness/contrast cause differences in visual
  latency → a **Pulfrich effect** (false motion-in-depth).
- **No neural adaptation** appeared over 7-day periods, possibly because the
  dominant eye's pupil constantly varies with illumination.
- Can distort perception of relative movement — a hazard, e.g. driving.

## Learnings for this project

- Important caveat for people with *partial* vision in the second eye (not
  full monocularity): unequal eyes can create a persistent driving hazard.
- Belongs in the driving guide as a specific, named risk.
