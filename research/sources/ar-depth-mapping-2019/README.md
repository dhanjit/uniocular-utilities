# Enhanced depth navigation through AR depth mapping (2019)

**Citation:** "Enhanced Depth Navigation Through Augmented Reality Depth
Mapping in Patients with Low Vision." *Scientific Reports*. 2019.
**Identifiers:** DOI 10.1038/s41598-019-47397-w.
**Access:** Open access (<https://www.nature.com/articles/s41598-019-47397-w>;
<https://pmc.ncbi.nlm.nih.gov/articles/PMC6677879/>).
**Verification:** From search digest.

## Scope

Tested a depth-to-pseudocolor AR overlay on a HoloLens as a navigation aid
for retinitis pigmentosa (RP) patients.

## Key points

- RP patients navigated a maze and avoided obstacles ~**50% better** with the
  aid.
- A **color-mesh overlay** preserves true color/texture through gaps, unlike
  full-VR replacement.

## Learnings for this project

- Proof that AR depth rendering improves real-world mobility — relevant to a
  uniocular **depth-cue rendering** module and blind-side assist.
- **Caveat:** subjects were low-vision (RP), not uniocular with good acuity;
  for our users the value is depth/blind-side rendering, *not* magnification
  or scene narration.
