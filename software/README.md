# Software utilities

Runnable tools for uniocular people — preferably installable-free and
usable on a phone, desktop, or browser.

See the [roadmap](../ROADMAP.md) for the planned utilities. Current
candidates:

- **Blind-side assist** — surface what is on the non-seeing side with
  audible/haptic alerts.
- **Distance estimator** — estimate distance from monocular camera cues.
- **Depth-cue trainer** — exercises that strengthen monocular depth cues.
- **Stereo-to-mono converter** — flatten 3D/stereoscopic media.
- **Eye-strain manager** — break reminders and reading-comfort presets.

Each tool lives in its own subdirectory with a README that states what it
does, how to run it, and which roadmap item it implements. Tools must
follow the project [accessibility guidelines](../specifications/accessibility-guidelines.md).
