# Manuals and guides

Plain-language, task-oriented documentation for living with monocular
vision.

See the [roadmap](../ROADMAP.md) for planned guides. Current candidates:

- **Living with one eye — first 90 days** (flagship guide).
- Daily-living guides: pouring, cooking, stairs, crowds, doorways.
- Driving with monocular vision: mirror setup, scanning habits, regional
  legality notes.
- Sports and hobbies: catching, cycling, swimming adaptations.
- Returning to work and school: workspace setup, requesting adjustments.

Each guide is a Markdown file. Guides must follow the evidence and
language principles in [CONTRIBUTING.md](../CONTRIBUTING.md): cite sources,
flag anecdotal advice, and make clear this project does not replace
medical guidance.
