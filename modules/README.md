# Helping modules

Reusable, embeddable components that other projects can drop in.

See the [roadmap](../ROADMAP.md) for planned modules. Current candidates:

- **Depth-cue rendering library** — shadows, parallax, and size scaling
  that make depth readable with one eye.
- **Alerting module** — combined audio and haptic alerts for proximity
  events.
- **Reference-size dataset** — known object dimensions for distance
  estimation.

Each module lives in its own subdirectory with a README covering its API,
dependencies, and the roadmap item it serves. Modules should be small,
dependency-light, and usable on their own.
